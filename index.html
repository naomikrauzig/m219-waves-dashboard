RV Meteor M219 WAVES

M219 WAVES Cruise Dashboard

A GitHub Pages-ready dashboard for rolling Copernicus Marine forecast maps from today through five days ahead, satellite-derived maps, in-situ observations, and derived diagnostics along the planned M219 cruise corridor.

Photo: Mario Mueller / GEOMAR

Planned cruise 30 May - 28 Jun 2026 Last updated: pending first data run

Product Availability

Rows are selectable layers; columns are the rolling daily map window.

Datasets Used

Dataset IDs are ready for the Copernicus Marine Toolbox and can be updated in data/products.json.

Layer Roadmap

Assessment of what can be automated for cruise operations.

Modeled Forecast

Operational now for five-day forecast maps: waves, currents, temperature, and salinity.

Satellite Maps

Feasible for latest available days, but not true forecasts; latency and clouds affect SST and chlorophyll.

Argo And Drifters

Feasible as point, profile, and trajectory overlays using Copernicus in-situ and NOAA GDP sources.

Derived Diagnostics

Feasible as computed products: Ekman pumping, upwelling proxies, eddy kinetic energy, and eddy masks.

Operations Notes

  • The route currently uses planning waypoints. Replace data/route.geojson with DShip, AIS, or station positions when available.
  • The update workflow expects Copernicus Marine credentials in GitHub repository secrets and runs daily.
  • Generated plots should be written to assets/snapshots; the page automatically prefers real images over placeholders.
