M219 WAVES Cruise Dashboard

A GitHub Pages-ready dashboard for rolling Copernicus Marine and ERA5 quick-look maps along the planned M219 WAVES cruise corridor, including model forecasts, latest available satellite-derived products, atmospheric forcing, and derived diagnostics.

Product Availability

Rows are selectable layers; columns show the rolling map window and latest available source dates where relevant.

Datasets Used

Dataset IDs are configured for the Copernicus Marine Toolbox and CDS API and can be updated in data/products.json.

Layer Roadmap

Assessment of what can be automated for cruise operations.

Modeled Forecast

Operational for rolling forecast maps: waves, surface currents, TEOS-10 Conservative Temperature, and Absolute Salinity.

Satellite Maps

Operational for latest available near-real-time SLA/ADT, sea surface salinity, and chlorophyll-a products; latency depends on product availability.

Argo and Drifters

Planned as point, profile, and trajectory overlays using Copernicus in-situ and NOAA GDP sources.

Derived Diagnostics

Operational for ERA5-based wind stress and Ekman pumping; planned extensions include upwelling proxies and eddy diagnostics.

Operations Notes

  • The route currently uses planning waypoints. Replace data/route.geojson with DShip, AIS, or station positions when available.
  • The update workflow expects Copernicus Marine and CDS API credentials in GitHub repository secrets and runs daily.
  • Generated maps are written to assets/snapshots and listed in data/manifest.json.
  • Model temperature and salinity maps are converted to TEOS-10 Conservative Temperature and Absolute Salinity before plotting.
